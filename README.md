# BTPowerNotice
支持在托盘图标中显示蓝牙电量

<img width="444" height="282" alt="image" src="https://github.com/user-attachments/assets/7bff3a46-861d-4956-8d07-b488cbed9e7b" />

## 功能

实施显示已连接的设备电量信息

安装库

```
pip install -r requirements.txt
```



打包命令

```
pyinstaller -F -w -i .\icon\icon.ico --version-file=version_info.txt main.py
```
